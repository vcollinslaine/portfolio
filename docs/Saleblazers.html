About

    Saleblazers is an open-world, multiplayer shopkeeping game.

    I interned at Airstrafe for 11 weeks, both
in-person and remote.

    As a technical game design intern, I ideated, prototyped, and implemented new features.

    I also created tools and improved pipelines to speed up other designers' iteration time.

    Saleblazers is launching in Fall 2023 on Steam.

A gif showing the status UI I created (top left), as well as functionality for removing the burn attribute by rolling.

Design

  • Wrote design documents for new systems, quests, and encounters
  • Participated in larger player experience and progression playtests and design decisions
  • Created visual mockups of UI elements and implemented them in the Unity UI system
  • Added npcs and wrote dialogue to improve new player experience
  • Made pricing and economy decisions for balancing items and weapons

Implementation & Tools

  • Overhauled attribute system to improve scalability and iteration time
  • Modified how data was stored and serialized to increase data entry efficiency
  • Implemented flexible weather system with zones in the open world
  • Prototyped new features and systems for designers to test with and iterate on
  • Integrated changes into a pre-existing, complex, networked codebase
